1. <html>:

    The root element of an HTML page.

  2. <head>:

    Contains meta-information about the HTML document.

  3. <title>:

    Sets the title of the HTML document.

  4. <div>:

    A generic container element used to group other elements.

  5. <span>:

    An inline container often used for applying styles or scripting.

  6. <input>:

    Creates an input field, commonly used within forms.

  7. <form>:

    Defines an HTML form for user input.

  8. <button>:

    Creates a clickable button.

  9. <label>:

    Defines a label for an input element.

  10. <select>:

    Creates a drop-down list.

  11. <option>:

    Defines an option within a <select> element.

  12. <textarea>:

    Defines a multiline text input control.

  13. <strong>:

    Represents strong importance or emphasis.

  14. <em>:

    Represents emphasized text.

  15. <br>:

    Inserts a line break.

  16. <hr>:

    Represents a thematic break or horizontal rule.

  17. <table>:

    Defines a table.

  18. <tr>:

    Defines a table row.

  19. <td>:

    Defines a table cell.

  20. <th>:

    Defines a header cell in a table.

  21. <pre>:

    Defines preformatted text, preserving both spaces and line breaks.

  22. <code>:

    Represents a piece of code.

  23. <abbr>:

    Defines an abbreviation or acronym.

  24. <blockquote>:

    Defines a block of text that is a quotation from another source.

  25. <cite>:

    Defines the title of a creative work, such as a book or movie.

  26. <dfn>:

    Defines a definition term.

  27. <mark>:

    Represents text highlighted for reference or notation.

  28. <time>:

    Represents a specific period in time.

  29. <progress>:

    Represents the completion progress of a task.

  30. <meter>:

    Represents a scalar measurement within a known range.
